# GIS Workshop

LASTRAN, UFRGS, Porto Alegre, 2026

Introdução à análise de dados geoespaciais com Python

O documento `workshop.ipynb` é um notebook de Jupyter que contém uma introdução à análise de dados geoespaciais usando Python. Ele inclui exemplos de código para manipulação de dados tabulares com `pandas` e para trabalhar com dados geoespaciais usando `geopandas`.

A forma recomendada de executar o notebook é com Google Colab, que é uma plataforma gratuita de notebooks Jupyter hospedada na nuvem. Para abrir o notebook no Google Colab, siga os passos abaixo:

1. Acesse o repositório do workshop no GitHub: [https://github.com/simovilab/lastran-workshop-gis](https://github.com/simovilab/lastran-workshop-gis)
2. Clique no arquivo `workshop.ipynb` para visualizá-lo.
3. Acesse o Google Colab em [https://colab.research.google.com/](https://colab.research.google.com/).
4. No Google Colab, clique em "File" (Arquivo) > "Open notebook" (Abrir notebook).
5. Na aba "GitHub", cole a URL do repositório do workshop e pressione Enter.
6. Selecione o arquivo `workshop.ipynb` na lista de arquivos e clique em "Open" (Abrir).
7. O notebook será aberto no Google Colab, onde você poderá executar as células de código e seguir o conteúdo do workshop.
